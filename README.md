## Getting Started

### Installation

1. Get the Flutter SDK [https://docs.flutter.dev/get-started/install/windows](https://docs.flutter.dev/get-started/install/windows)
2. Clone the repo
   ```sh
   git clone https://github.com/hboudanes/pharmacie_de_garde_UI
   ```
3. Install packages
   ```sh
   flutter pub get
   ```
